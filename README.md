# Bank Management System

A full-stack desktop application designed to automate banking operations and manage user accounts with a relational database backend.

## 🚀 Features
- **Secure User Authentication:** Login system for customers.
- **CRUD Operations:** Create, Read, Update, and Delete account information in real-time.
- **Transaction Management:** Handle deposits, withdrawals, and balance inquiries.
- **Relational Database:** Structured MySQL backend to ensure data integrity.

## 🛠 Tech Stack
- **Language:** Python 3.x
- **Database:** MySQL
- **GUI Framework:** Tkinter
- **Database Connector:** `mysql-connector-python`

## 📋 Architecture
The application follows a two-tier architecture:
1. **Front-end:** A Tkinter-based GUI providing a user-friendly interface for data entry.
2. **Back-end:** A Python logic layer that processes business rules and communicates with the MySQL server via SQL queries.
